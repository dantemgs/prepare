10. Перечислить по пять знакомых вам псевдоклассов и псевдостилей с указанием их функционала.

Псевдоклассы:

  • :hover - стили для элемента, на который навели курсор мыши
  • :focus - стили для элемента, находящегося в фокусе
  • :active - стили для активной ссылки
  • :visited - стили для ссылок, уже посещенных пользователем
  • :first-child - стили для первого ребенка элемента

Псевдоэлементы:

  • ::before - для отображения контента до элемента селектора
  • ::after - для отображения контента после элемента селектора
  • ::first-line - стили для первой строки
  • ::first-letter - стили для первого символа
  • ::selection - стили к выделенному пользователем тексту
